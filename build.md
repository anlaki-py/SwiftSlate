# Building SwiftSlate

## Requirements
- Android SDK
- JDK 17+

## Build Commands

**Debug APK:**
```bash
bash gradlew assembleDebug
```

**Release APK:**
```bash
bash gradlew assembleRelease
```

## Output
APKs are generated in `app/build/outputs/apk/`:
- Debug: `debug/app-debug.apk`
- Release: `release/app-release.apk`
